# ADR 0007: Recovery Dual-Mode Product

Status: Accepted

Date: 2026-07-10

## Context

The pre-recovery project accumulated broad game features before its core water behavior, architecture, and evidence met a trustworthy product bar.

## Decision

One deterministic simulation core will support two equal clients: a narrow player-facing sandbox and an engineering laboratory. The first recovery release contains the pour/build/reset/save basin loop plus canonical lab scenarios. Secondary devices, objectives, progression, broad gallery content, and decorative polish remain outside the release UI until separately accepted.

## Consequences

- Product acceptance requires both numeric evidence and human review where applicable.
- Sandbox UI stays player-facing; engineering controls belong in lab mode.
- Feature count is not a completion metric.
